前言

CSS3是层叠样式表的最新标准，有许多新特性，详细参考菜鸟教程

用CSS3绘制一个立方体可以有助于我们很好地构建立体感和空间感，对加深CSS3的旋转和位移两个属性有更深的理解和认识。

内容核心

绘制html结构

一个立方体有6个面，而不同的面是相对于一个参考而规定的，所以我们需要一个“参考”来使得6个面能在这个立方体的不同位置。

我们用ul-li结构来绘制（也可以用div包着6个div），ul作为父元素可以作为“参考”，指引6个li作为不同的面。

html

添加CSS属性

父元素

我们给父元素ul（父元素div）加上3D属性’transform-style: preserve-3d’,并根据需要的立方体大小给它长宽，当然可以设置margin使得立方体处于中间位置

css

子元素

子元素的长宽是基于父元素的，这里设置了200像素，所以我们设置200px，然后我们借助’transform’的三个属性（rotateX、rotateY、translateZ）来变换位置并给不同面不同的背景颜色作为区别，为了增加立体感，我们给所有面都赋予0.4的透明度。

css
